# README



### backend 실행

##### 가상환경 설치 방법

```python 
python -m venv venv
```

##### 가상환경 들어가는  법

```python 
source venv/Scripts/activate
```

##### 가상환경 끄는법

```python 
deactivate
```

##### 패키지 다운로드

```python 
pip install -r requirements.txt
```



##### db tags 저장

```python
python manage.py loaddata data/tags.json
```
